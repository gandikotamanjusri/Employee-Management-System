# Employee Management System (Java)

A console-based Employee Management System built using Java Collections and File Handling.
This project demonstrates CRUD operations, searching, reporting, and file persistence using serialization.

---

## 📌 Features
- Add, view, update, and delete employees (CRUD)
- Search employees by department
- Generate salary and department reports
- Save employee data to file
- Load employee data from file
- Uses ArrayList and HashMap for efficient storage

---

## 🛠 Technologies Used
- Java
- Collections (ArrayList, HashMap)
- File Handling (ObjectInputStream, ObjectOutputStream)
- Date & Time API
- Exception Handling
